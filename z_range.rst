.. _z_range:

================
Range (function)
================

.. function:: Array Range( Number inicio, Number fim )

Essa função retorna um array numérico com os valores de ``inicio`` até ``fim``. Costuma ser abreviada pela sintaxe definida pelo pré-processador ``@{ inicio .. fim }``.
