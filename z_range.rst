.. _z_range:

================
Range (function)
================

.. function:: Array Range( Number inicio, Number fim )

Recebe um parâmetro numérico de início e outro de fim e retorna um array com os valores dentro do dado intervalo.
Costuma ser abreviada pela sintaxe definida pelo pré-processador ``@{ inicio .. fim }``.
